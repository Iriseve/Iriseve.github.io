2. 求两个正整数的最大公约数#

方法二：辗转相减法#

辗转相减法（求最大公约数），即尼考曼彻斯法，其特色是做一系列减法，从而求得最大公约数。例如 ：两个自然数35和14，用大数减去小数，(35,14)->(21,14)->(7,14)，此时，7小于14，要做一次交换，把14作为被减数，即(14,7)->(7,7)，再做一次相减，结果为0，这样也就求出了最大公约数7

#include<iostream>

int main(){
    long m,n,r,max,min;
    scanf("%ld %ld",&m,&n);
    max = m>=n?m:n;
    min = m<=n?m:n;
    while(max-min!=0){
        r=max-min;//差
        max  = r>=min?r:min;
        min = r<=min?r:min;
    }
    printf("%ld",min);
}

NC13134 牛牛的数列#

牛牛的数列 (nowcoder.com)

主要思想#

  • 以断点为关注点，观察它的前面和后面最长连续上升序列长度。

  • 动态规划思想

思路#

left[i]表示从左往右以a[i]为终止元素的最长连续上升序列的长度。

由于当以a[i]为终止元素时，从左往右可以利用left[i-1]，避免一次又一次重复的比较。

right[i]表示从右往左以a[i]为起始元素的最长连续上升序列的长度。

当以a[i]为起始元素时，如果从左往右，则每一次找其right[i]都需要不断的把后面所有数遍历即(n-1)!，而从右往左则只需比较相邻的，然后利用right[i+1]递归。

先用两次循环求出left[i]和right[i]；

改变数字(假设这个数字是断点，即不管实际是否连续）

xxx中间数是断点，则改变原始长度只有三种，

要么三个拼一起a[i + 1] - a[i - 1] >= 2

要么中间的和前面在一起a[i - 1] < 1000000000

要么中间的和后面的在一起a[i + 1] >= 2
